# Lógica de Programación 5

## Descripción

Programa realizado en Java que funciona como un mini diccionario Español-Inglés.

El sistema:

- Guarda palabras en un HashMap.
- Selecciona 5 palabras aleatorias.
- Solicita al usuario su traducción en inglés.
- Verifica si las respuestas son correctas.
- Muestra resultados finales.

---

## Tecnologías utilizadas

- Java
- IntelliJ IDEA
- HashMap
- ArrayList
- Scanner

---

## Ejecución

1. Abrir el proyecto en IntelliJ IDEA.
2. Ejecutar la clase `Main.java`.
3. Responder las preguntas en consola.

---

## Ejemplo de ejecución

```text
Traduce la palabra: gato
cat
Correcto

Traduce la palabra: naranja
oranje
Incorrecto
La respuesta correcta es: orange
```
